Distributed Systems and Networks

FastTree

Final Year Project, Systems Research Group, Microsoft Research Asia

A distributed machine learning framework based on Gradient Boosting Decision Tree (GBDT, GBRT, GBM or MART). The current version is built on top of ChaNa, the RDMA-optimized distributed computing engine developed by MSRA Systems Group. With both system-level and algorithm-level optimizations, memory usage and communication cost are reduced, and the performance is better than popular similar tools (e.g. XGBoost).

I have been individually building the framework on ChaNa. However, the techniques we proposed are general, and have been implemented in Microsoft's other opensource GBDT frameworks: LightGBM (received 2000+ stars on GitHub), SparkTree (Spark version of FastTree). Our framework on ChaNa is seamlessly integrated into the AllReduce-based application module of ChaNa, enjoying the NUMA-aware architectural design and RDMA networking optimizations of ChaNa, while introducing almost no system overhead thanks to ChaNa's very low-level abstractions. My undergraduate thesis is based on this project.

Compressed Online Media Streaming with Dynamic Huffman Tree

Software Design and Practice I, Harbin Institute of Technology

The Dynamic (or Adaptive) Huffman Encoding algorithm is an online algorithm that can compress and decompress streamed data without knowing the whole data in prior. This is exactly the case in networked media streaming: media file is transferred while played. We design and implement a local environment which simulates real networking environments and do compressed transmission with dynamic huffman encoding. It also deals with out-of-order packets with a priority-queue-controlled stream buffer.

SINH Interpreter in Haskell

COMP3259 Principles of Programming Languages, The University of Hong Kong

We design and develop interpreters for both functional and imperative SINH in the functional programming language Haskell.

View Project

SINH is a simple functional script language introduced in the course lectures. It supports first-class functions, recursion and several types of error checking. It also has another version supporting imperative features like mutable variables.

Features:

  • First-Class Functions
  • Recursion
  • Error Checking (e.g. static type checking, division-by-0 checking)

For the imperative version, we implement mutable-state variables with simple memory management using Monads.

Computer Architecture and Embedded Systems

CPU on FPGA

Summer Project, Harbin Institute of Technology

It is a single-cycle four-beat CPU with multiple interrupt support on FPGA board. I wrote it in the VHDL language, and adapted it to the Digilent Nexys 3 FPGA board. I divided the memory address space into two parts (code segment and data segment). Valid binary programs can be downloaded into the memory and ran with full access to the whole memory.

We also designed a toy on this board to demonstrate how multiple interrupt works. Its code repository is here, and video is here.

View Project Demo Video

This "mini CPU" includes an Intel-8259-like Multiple Interrupt Controller as a peripheral. With the INT instruction, the process can run into multiple-level interrupts. Interrupt service routines are preset in the beginning bytes of the memory.

To make the execution process easy to observe, I added a PmodLCD display, which shows each instruction being executed, realtime register contents and interrupt levels. The CPU's execution beats can be controlled with a button or the system clock on board.

Human Body Tracking on DJI M100 UAV (Ongoing)

Starred Project of Embedded System Principle and Practice, Harbin Institute of Technology

We install Raspberry Pi as the "brain" of the UAV to control its flight. With the help of the realtime video stream tracked from 8 cameras deployed at 4 sides, we make the drone follow a designated human body. I am responsible for sending accurate motion controlling instructions with Raspberry Pi over UART.

This course project later became an National Innovation Project in HIT. It is still in progress as some new members have joined.
